Integra-7 Cubase Patch Script Builder

By Greg Simon

This web application will build an inventory of all addressable patches in a connected Integra-7 sound module. Any User patches, Preset patchces, and SRX cards currently loaded. This list will then be sorted by category and a Cubase patch script will be downloaded. You can follow instructions in your Cubase manual for how to install this script.

Limitations

  • The ExPCM data is *not* addressable via SysEx so will not be inventoried.
  • Any SRX cards that are not currently playable (i.e., not "loaded" into your Integra-7 will not be addressable and not included in this patch script.)

MIDI Input: MIDI Output:
You must have MIDI enabled on this version of Chrome. You can do this by entering chrome://flags in your URL bar and searching for "Enable Web MIDI API" Enable it, then relaunch chrome.
